Real-time Website Stats

You're viewing a demo site created for Jackalope, a real-time web stats viewing application entry for the Node Knockout competition.

Every movement you make with your mouse is instantly transferred back through our Node.js server and back to the admin view, generating a heatmap in real-time for the admin to view!

Can I try it?

Jackalope displays live data, you can try generating data right now! Leave this page open, then Click Here to launch http://website.com in a new browser window. Move and click your mouse on http://website.com and watch your activity be displayed on the heatmap.

How does it work?

Visitor mouse data is collected and sent via WebSocket through a Node.JS server. This page has another WebSocket connection where it receives data and displays it.

Learn more

Jackalope is a hot new way to see visitors interacting with your site in realtime. Currently in beta.

Jackalope requires a browser that supports WebSocket, such as Safari 5, and currently supports visualization and data gathering from up to 5 concurrent users.
